The setup phase confirms that the monitoring environment is functional.

- The Wazuh server is running and accessible via the dashboard.
- The Windows 11 endpoint is configured with the Wazuh agent.
- The agent is connected and reporting events successfully.
- Sysmon is installed on the endpoint to provide enhanced visibility
  into process creation and network activity.

This setup ensures that events generated on the endpoint
can be collected and analyzed in the SIEM.
